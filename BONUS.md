# BONUS.md — 가산점 신청

## A. AI Agent / 스킬 / 워크플로우 적극 활용 (+1점)

**사용 도구:** Claude Code (CLI)

**활용 사례:**
- 프로젝트 문서 전체(`.planning/`, `docs/`, `AGENTS.md`, `README.md`)를 Claude Code로 자동 생성
- 코드 작성 시 Claude Code를 사용해 컴포넌트 구조 설계 및 API 연동 코드 생성
- 발표 자료 초안을 Claude Code로 생성 후 직접 검토·수정

**절약된 시간:** 문서 작성 약 3~4시간 → 30분으로 단축
