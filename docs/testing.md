# 테스트 전략

## 테스트 유형

### 1. 수동 기능 테스트 (주요)

발표 전 반드시 아래 시나리오를 직접 실행해서 확인한다.

| 시나리오 | 기대 결과 | 확인 |
|---|---|---|
| 비빔밥 사진 업로드 | "비빔밥" 인식 + 칼로리 표시 | |
| 된장찌개 + 밥 함께 촬영 | 2개 음식 각각 분리 인식 | |
| 음식명 수정 후 저장 | 수정된 이름으로 홈에 표시 | |
| 앱 종료 후 재실행 | 오늘 식단 데이터 유지 | |
| 비행기 모드에서 사진 업로드 | 에러 메시지 표시 (앱 크래시 없음) | |
| 하루 3끼 입력 후 칼로리 합계 | 각 칼로리의 합과 일치 | |
| 주간 리포트 탭 이동 | 최근 7일 차트 표시 | |
