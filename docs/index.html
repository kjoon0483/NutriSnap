NutriSnap

사진 한 장으로 음식을 인식하고, AI가 칼로리와 영양소를 분석해주는 모바일 앱

⚠ 문제 정의

01

기록이 귀찮다

음식명과 양을 매번 직접 입력해야 해서 지속하기 어렵다

02

검색이 번거롭다

칼로리를 일일이 검색해야 해서 꾸준한 식단 관리가 힘들다

03

균형 파악이 어렵다

영양 불균형 여부를 직관적으로 확인할 방법이 없다

✦ 비전 & 솔루션

사진 한 장 → 자동 분석 → 식단 기록 완료

카메라로 음식을 찍으면 Claude Vision AI가 자동으로 인식하고 칼로리·영양소를 계산합니다.
사용자는 결과를 확인하고 수정하기만 하면 됩니다.
